## What Is It?

I'm building an interactive music guessing game where users identify songs from 5-second audio clips. The game will feature different music categories and integrate with the Deezer API to access a diverse music library.  

It's a web-based game that showcases:
- API integration  
- Audio handling  
- Interactive frontend development

This adds an interactive, playable element to my portfolio that stands out from static projects.

It demonstrates:

## Fears, Uncertainties, and Doubts

- **API Rate Limits:** Deezer might limit requests during gameplay.  
- **Audio Playback Issues:** Different browsers (especially iOS Safari) handle audio differently.  
- **Difficulty Balancing:** Five seconds might be too easy or too hard depending on the song.  
  - I may need to select specific song portions (chorus vs. intro)  
  - Or add difficulty levels
